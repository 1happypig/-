# UE5 白膜搭建工作流

> 从零搭建一个可游玩白膜关卡的标准化流程。配合 [[引擎使用/ue5快捷键]] 使用。

---

## 总览

```
前期准备 → 铺地平面 → 搭主体结构 → 开门窗通道 → 放置掩体 → 跑测迭代 → 最终检查
```

---

## 模式与工具速查

> 白膜阶段在不同模式之间频繁切换，先搞清楚每个模式干什么。

| 快捷键 | 模式 | 白膜中什么时候用 |
|--------|------|-----------------|
| **Shift + 1** | Place / Select（放置 / 选择） | **默认模式**，选物体、W/E/R 变换 |
| **Shift + 5** | Geometry Editing（几何体编辑） | 编辑 BSP 面 / 边 / 顶点，拉造型 |
| **Alt + P** | Play / 运行游戏 | 跑测 |
| **ESC** | 退出运行，回编辑模式 | 跑测结束 |
| **F8**（运行时） | 弹出玩家控制器（Eject） | 跑测中脱离角色，自由飞观察场景 |
| **Pause**（运行时） | 暂停游戏 | 冻结场景截图、测量、标注 |

> 💡 白膜最常用的切换节奏：**Shift+1 搭 → Shift+5 修 BSP → Shift+1 继续搭 → Alt+P 跑 → F8 观察 → ESC 退**

---

## 阶段 1：前期准备

- [ ] 打开 UE5，新建 **Basic** 或 **Empty** 模板关卡
- [ ] 导入人形比例模型（1.7-1.8m），拖入场景作为尺度参考
- [ ] 设置网格吸附单位：建议从 **100**（粗搭）→ **50**（细化）→ **10**（微调）逐步切换
- [ ] 在 World Settings 中设置 GameMode（如需要跑测）
- [ ] 保存关卡，命名规范：`地图名_版本号_日期`

> ⚠️ 比例模型是白膜阶段最重要的参照物，所有尺寸都以它为基准。

---

## 阶段 2：铺地平面

- [ ] 用 BSP Box 或 Plane 铺设**地面基底**
- [ ] 地面颜色统一用灰色，方便后续视觉区分
- [ ] 如果地图有高差，在这一步就拉出大致地形起伏
- [ ] 用 **[ ]** 快速切换网格单位，确保地平面尺寸为整数

> 💡 地面尽量用整十整百的尺寸（如 10000×10000），后续模块对齐会省很多事。

---

## 阶段 3：搭主体结构（墙体 + 楼层）

- [ ] 从 **核心 POI** 开始搭，再扩展到次要区域
- [ ] 墙体高度 = 比例模型高度的 **1.5-2 倍**（约 2.5-3.5m）
- [ ] 墙体颜色用**浅色**，与灰色地面形成对比
- [ ] 多层建筑：先铺完整层地板，再搭下层墙体，再上层——**一层一层来**
- [ ] 每个房间/区域搭完后，按 **End** 让结构自动落地
- [ ] 边搭边用 **右键+WASD** 飞行检查空间尺度是否舒适

### BSP 造型编辑（Shift + 5）

> 搭好基础方块后，切到几何编辑模式精修造型。

| 操作 | 方法 |
|------|------|
| 进入编辑模式 | **Shift + 5** |
| 拉伸面 | 选中面 → **W** 拖拽 |
| 移动边 | 选中边 → **W** 拖拽 |
| 移动顶点 | 选中顶点 → **W** 拖拽 |
| 吸附编辑 | 拖拽时按住 **Ctrl** |
| 返回放置模式 | **Shift + 1** |

### 搭墙效率技巧

| 技巧 | 操作 |
|------|------|
| 复制墙体 | **Alt + 拖拽** 或 **Ctrl + D** 原地复制 |
| 精确对齐 | 拖拽时按住 **Ctrl** 吸附网格 |
| 顶点对接 | 按住 **V** 拖拽，吸附到目标顶点 |
| 快速定位 | 选中墙面，按 **F** 聚焦 |
| 切换网格精度 | **[** 减小 / **]** 增大吸附单位 |

---

## 阶段 4：开门窗与通道

- [ ] 用 BSP 减法（Subtractive）或直接调整 BSP 尺寸来开洞
- [ ] 门宽 ≥ 比例模型宽度的 **2 倍**（约 1.6-2m）
- [ ] 走廊宽度 ≥ 比例模型宽度的 **2-2.5 倍**（约 2-3m）
- [ ] 每个房间至少保留 **2 个出入口**（主要入口 + 侧翼/窗户可翻越）
- [ ] 检查所有门洞高度：角色是否能正常通过？是否需要跳跃/蹲下？

> ⚠️ 搜打撤地图注意：不要所有房间都只有一个门——那会变成"谁先进谁死"。

---

## 阶段 5：放置掩体

- [ ] 先在 **主要交战区** 放置掩体（走廊转角、大厅中心、门口两侧）
- [ ] [[专业词汇注解/全掩体|全掩体]] 用 **深色**，[[专业词汇注解/半掩体|半掩体]] 用 **中间色**

### 掩体分布自检

- [ ] 每个主要交战区至少 **2 个掩体群**
- [ ] 掩体之间间距合理，不会让玩家在无掩护状态下跑超过 **2-3 秒**
- [ ] 高位（窗口/楼梯口）和低位之间有掩体对应的视野关系
- [ ] 掩体不会完全封死一条路径（至少留侧身/投掷物空间）
- [ ] 存在 [[专业词汇注解/可穿透掩体|可穿透掩体]]（薄墙/木箱）供信息博弈

---

## 阶段 6：跑测与迭代

> 这是白膜阶段**最重要的环节**——搭完不等于做完。

### 跑测标准操作流

> 记住这个节奏，每次跑测都用：

```
Alt+P 启动游戏 → 按玩家视角走 → 到关键位置按 F8 弹出
→ 自由飞到空中俯瞰 → 按 Pause 暂停截图
→ 按 F8 回到角色 → 继续跑下一段 → ESC 退出
```

### 跑测核心快捷键

| 快捷键 | 功能 | 使用场景 |
|--------|------|---------|
| **Alt + P** | 启动运行 | 进跑测 |
| **F8**（运行时） | **弹出（Eject）**，脱离角色 | 飞到空中观察掩体分布、枪线、空间关系 |
| **Pause / Break** | 暂停游戏 | 冻结当前帧，截图标注问题位置 |
| **F8**（弹出后再按） | 回到角色身上（Possess） | 继续跑下一个位置 |
| **ESC** | 退出运行 | 回到编辑模式修改 |
| **F11**（运行时） | 沉浸模式，隐藏 UI | 截图展示用 |
| **Shift + F1**（运行时） | 释放 / 夺回鼠标 | 暂停时点选 UI 或截图 |

> 💡 跑测时最常见的组合：跑到 Choke 点 → **F8** 弹出 → 飞到对方一侧看枪线是否过强 → **Pause** 截图 → **F8** 回到角色 → 继续跑。

### 第一轮跑测（自己跑）

- [ ] 按玩家视角走完 **所有动线**：出生点 → 各 POI → 每个撤离点
- [ ] 检查：有没有"死路"？玩家会不会走到某处后不知道往哪走？
- [ ] 检查：[[专业词汇注解/Choke点|Choke 点]]是否有迂回路线？会不会变成"露头就死"？
- [ ] 检查：有没有某个位置视野覆盖太广（狙击位过强）或太窄（近战过于集中）？
- [ ] 用 **Ctrl+1~9** 保存关键问题位置的书签，方便定位修改

### 第二轮跑测（找别人跑）

- [ ] 找 1-2 个人跑你的白膜，**不给任何提示**，观察他们的行为
- [ ] 记录他们在哪里停下来犹豫（→ 动线不清晰）
- [ ] 记录他们在哪里走错路（→ 引导不够）
- [ ] 记录他们觉得"太大""太小""太绕"的地方

### 迭代修改

- [ ] 根据跑测反馈修改白膜
- [ ] 改完再跑一遍，确认问题已解决、没有引入新问题
- [ ] 重复此循环直到核心体验稳定

---

## 阶段 7：最终检查清单

搭建完成前，逐项确认：

### 空间尺度
- [ ] 所有门宽 ≥ 1.6m、走廊宽 ≥ 2m
- [ ] 没有"卡人"的狭窄缝隙
- [ ] 天花板高度充足，跳跃不会撞头

### 动线与引导
- [ ] 从每个出生点到每个撤离点都有至少 2 条路线可选
- [ ] 主要 POI 从远处能看到（[[专业词汇注解/地标|地标]]引导）
- [ ] 没有玩家会走进去却不知道怎么出来的区域
- [ ] Choke 点都有侧翼路线

### 掩体与战斗
- [ ] 每个交战区掩体充足且分布均衡
- [ ] 全掩体和半掩体颜色区分清晰
- [ ] 存在可穿透掩体供信息博弈

### 视觉清晰度
- [ ] 地面（灰）、墙体（浅）、掩体（深/中间色）三者颜色区分明显
- [ ] 关键路径和死胡同的视觉差异足够大

---

## 常见坑与对策

| 踩坑 | 症状 | 解决 |
|------|------|------|
| 先搭细节后定大结构 | 改了整体布局后细节全废 | **严格按阶段顺序来**，先大后小 |
| 不用比例模型 | 搭完发现门太窄、房间太小 | 第一步就放比例模型，全程对照 |
| 不设网格吸附 | 模块之间有缝隙、不对齐 | 全程按住 **Ctrl** 吸附网格 |
| 不会切 BSP 编辑模式 | 只能堆方块，做不出异形结构 | 熟悉 **Shift+5**，学会拖面/边/顶点 |
| 跑测不知道 F8 弹出 | 卡在角色视角，看不到全局空间关系 | 跑到位置就 **F8** 飞到空中看 |
| 只跑一遍就过 | 自以为没问题，别人一跑全是坑 | 至少跑 **3 轮**（自己→别人→改后再跑） |
| 掩体后补 | 搭完结构发现没地方放掩体 | **掩体和结构同时考虑**，不要最后才塞 |
