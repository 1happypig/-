# UE5 白膜搭建快捷键

> 白膜（Whitebox / Blockout）阶段快捷键速查。按使用场景分类，贴合实际工作流。

---

## 模式切换

> 白膜阶段最常驻的模式是 **Place Mode（放置模式）** 和 **Geometry Editing Mode（几何编辑模式）**。

| 快捷键 | 模式 | 白膜阶段的用途 |
|--------|------|---------------|
| **Shift + 1** | Place / Select（放置 / 选择） | **默认模式**，选择和摆放物体 |
| **Shift + 2** | Landscape（地形） | 捏地形高差 |
| **Shift + 3** | Foliage（植被） | 白膜阶段一般不用 |
| **Shift + 4** | Mesh Paint（网格绘制） | 批量摆放重复模块 |
| **Shift + 5** | Geometry Editing（几何体编辑） | **编辑 BSP 顶点 / 边 / 面** |
| **Shift + 6** | Fracture（破坏） | 白膜阶段一般不用 |
| **Shift + 7** | Layers（图层） | 管理关卡分层 |

---

## 编辑模式（Place Mode）下的操作

### 变换工具

| 快捷键 | 功能 |
|--------|------|
| **Q** | 选择模式（取消变换工具，自由选择） |
| **W** | 移动（Translate） |
| **E** | 旋转（Rotate） |
| **R** | 缩放（Scale） |
| **Space** | 循环切换 W → E → R |

### 复制与删除

| 快捷键 | 功能 |
|--------|------|
| **Alt + 拖拽** | 复制 Actor 并移动到新位置 |
| **Ctrl + W**（移动时按住） | 复制并移动（同 Alt 拖拽，另一种方式） |
| **Ctrl + D** | 原地复制 |
| **Delete** | 删除选中物体 |

### 吸附（Snap）

> 白膜阶段全程按住吸附键，确保模块严丝合缝。

| 快捷键 | 功能 |
|--------|------|
| **Ctrl + W/E/R**（拖拽时按住） | 变换时吸附网格 |
| **V**（拖拽时按住） | 顶点吸附（Vertex Snap），对齐到目标顶点 |
| **[** | 减小网格吸附单位 |
| **]** | 增大网格吸附单位 |
| **End** | 将选中 Actor 吸附到下方表面（落地） |
| **Ctrl + End** | 将选中 Actor 的 Pivot 吸附到下方表面 |

## 视角控制

### 自由飞行（编辑模式下）

| 快捷键 | 功能 |
|--------|------|
| **右键按住 + WASD** | 自由飞行（Fly），白膜阶段最常用的移动方式 |
| **右键按住 + Q** | 垂直下降 |
| **右键按住 + E** | 垂直上升 |
| **右键按住 + C** | 缩小 FOV（拉远） |
| **右键按住 + Z** | 放大 FOV（拉近） |
| **鼠标滚轮** | 前后推拉视角 |
| **右键按住 + 滚轮** | 调整飞行速度 |
| **Shift + 拖拽** | 视角移动加速 |

### 视角定位

| 快捷键 | 功能 |
|--------|------|
| **F** | 聚焦选中物体 |
| **Ctrl + 数字键（1-9）** | 保存当前视角为书签 |
| **数字键（1-9）** | 跳转到已保存的书签 |
| **Alt + 左键拖拽** | 围绕点击位置旋转视角 |
| **Alt + 右键拖拽** | 围绕点击位置缩放视角 |

## 运行时（Play Mode）快捷键

> 白膜跑测的核心：**怎么跑、怎么停、怎么跳出角色自由观察。**
> 
> 运行方式：工具栏点击 **Play** 按钮，或快捷键。

### 启动与停止

| 快捷键 | 功能 |
|--------|------|
| **Alt + P** | 运行游戏（Play In Editor） |
| **F8**（编辑模式下） | 在视口中运行（Play In Viewport） |
| **ESC** | 停止运行，返回编辑模式 |
| **F4** | 关闭当前运行的视口 |

### 运行时观察场景

> 这是"我怎么在运行时停下来观察场景"的核心答案。

| 快捷键 | 功能 | 使用场景 |
|--------|------|---------|
| **F8**（运行时按下） | **弹出玩家控制器（Eject）** | 角色站在原地，你自由飞出去观察周围——检查掩体分布、枪线、Choke 点——**白膜跑测最常用的快捷键** |
| **Pause** / **Break** | 暂停游戏 | 冻结当前帧，方便截图标注问题位置 |
| **F8 后再按 F8** | 回到玩家控制器（Possess） | 飞完之后回到角色身上继续跑 |

> 💡 **白膜跑测标准操作**：Alt+P 启动 → 跑到关键位置 → 按 **F8** 弹出 → 飞到空中观察空间关系 → 按 **Pause** 暂停截图 → 按 **F8** 回到角色继续跑。

### 运行时视角

| 快捷键 | 功能 |
|--------|------|
| **F11**（运行时） | 沉浸模式（隐藏 HUD 和编辑器 UI） |
| **~** 或 **`** | 打开控制台（输入调试命令） |
| **Shift + F1** | 释放/夺回鼠标光标 |

## 编辑器界面控制

| 快捷键 | 功能 |
|--------|------|
| **G** | 游戏视图（Game View）——隐藏所有编辑器图标/网格/辅助线，预览实际效果 |
| **F11** | 沉浸模式——全屏编辑器，最大化视口空间 |
| **Ctrl + Space** | 打开内容侧滑菜单（Content Drawer） |
| **Ctrl + P** | 快速搜索资产（Asset Picker） |
| **Ctrl + Shift + S** | 保存所有 |
| **Ctrl + Z** | 撤销 |
| **Ctrl + Y** | 重做 |
| **Ctrl + F** | 在视口中搜索 Actor |
| **Ctrl + Shift + F** | 在内容浏览器中定位选中资产 |

## BSP 编辑模式（Shift + 5）

BSP 是白膜最常用的几何体工具：

| 快捷键 / 操作 | 功能 |
|---------------|------|
| **Ctrl + A**（Place Actors 面板） | 快速添加 Box Brush |
| **Shift + 5** | 进入 BSP 几何体编辑模式 |
| 选中面后 **拖拽** | 拉伸 / 推拉 BSP 面 |
| 选中边后 **拖拽** | 移动边，改变形状 |
| 选中顶点后 **拖拽** | 移动顶点，自由变形 |
| 按住 **Ctrl** 拖拽顶点/边 | 吸附网格编辑 |
| **Delete**（选中面/边/顶点） | 删除选中元素 |

## 按工作场景速查

### 搭结构时

```
Shift+1（放置模式）→ 选 Box Brush → W 移动 / E 旋转 / R 缩放
全程按住 Ctrl 吸附网格 ｜ 按住 V 顶点对齐 ｜ End 落地
Alt+拖拽 复制墙体 ｜ Ctrl+D 原地复制
```

### 编辑 BSP 造型时

```
Shift+5（几何编辑）→ 选面拖拽 → Ctrl+拖拽吸附 → Shift+1 回到放置模式
```

### 跑测时

```
Alt+P 运行 → 跑到关键位 → F8 弹出自由飞 → Pause 暂停看问题 → F8 回归 → ESC 退出
```

### 切换视角书签

```
Ctrl+1 保存当前位置 → 继续工作 → 按 1 秒回 → 再按 1 回到最新位置
```
